OS161 kernel

1. Synchronisation primitives : Implemented locks, condition variables and reader-writer locks.
2. System calls : open, close, read, write, lseek, dup2. 
3. Process support : fork, execv, getpid, waitpid, exit.
4. Virtual memory : Handling TLB faults by paging.
5. Swapping : Handling Page faults by swapping to disk.
